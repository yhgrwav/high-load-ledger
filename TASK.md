## Техническое Задание для High-Load Distributed Ledger Core

##  Цель проекта
- 1000-2000 rps на один инстанс
- покрытие логики современными технологиями (тесты, метрики, логи, трейсы, шардирование, кубер, дополнительный микросервис для генерации искусственной нагрузки и тд)

## 🛠 Технологический стек
- **Язык:** Go 1.25+
- **Взаимодействие:** gRPC (Protobuf)
- **Брокер сообщений:** Apache Kafka
- **Кэширование и Идемпотентность:** Redis
- **Хранилище данных:** PostgreSQL
- **Инфраструктура:** Docker, Docker Compose, Makefile
